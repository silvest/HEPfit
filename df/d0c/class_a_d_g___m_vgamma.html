Detailed Description

A class for the \(A_{\Delta\Gamma}\) parameter for \(M \to V \gamma\) decay.

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the \(A_{\Delta\Gamma}\) parameter for \(M \to V \gamma\) in terms of the helicity amplitudes \(H_V^{(+,-)},\overline{H}_V^{(+,-)}\), computed in the MVgamma class:

\[ A_{\Delta\Gamma} = \frac {2Re(q/p * ( H_V^{+*} \overline{H}_V^+ + H_V^{-*} \overline{H}_V^-))}{( |H_V^+|^2 + |H_V^-|^2 +|\overline{H}_V^+|^2 + |\overline{H}_V^-|^2)} \,. \]
