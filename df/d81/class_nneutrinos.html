Detailed Description

An observable class for the number of neutrinos obtained indirectly from the measurements at the Z pole.

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the number of neutrinos obtained indirectly from the measurements at the Z pole,

\[ N_{\nu} = \frac{\Gamma_\ell}{\Gamma_{\nu}}\left(\sqrt{\frac{12\pi R_\ell}{M_Z^2 \sigma_\mathrm{had}^0}}-R_\ell - 3\right)\,. \]
