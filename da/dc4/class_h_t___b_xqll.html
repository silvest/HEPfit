Detailed Description

A class for the binned observable \(H_T\) in \(B \to X_q l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the integral of \(H_T(q^2)\) in bins of \(q^2\) for \(B \to X_q l^+l^-\) :

\[ H_T[q_m^2,q_M^2] = \int{q_m^2}{q_M^2} dq^2 \int{-1}{+1} dz \frac{d^2\Gamma}{dq^2 dz^2} \frac{2}{3} P_0(z) + \frac{10}{3} P_2(z) \,. \]
