41 // These Higgs parameters are not exactly those of Eq. 4 of [2401.12279] but a small rotation of them
42 // These would be exactly the parameters of Eq. 4 of [2401.12279] in the basis in which Y3=0, which
43 // is the relevant basis to perform the matchings. Therefore, these wouldn't be the parameters or the
44 // Higgs basis but a small rotation of the Higgs basis in which Y3=0. The relation of these parameters
45 // and thos of the Higgs basis, expanding at order 1/Y2 would be:
46 // Y1 = Y1Higgs - Y3Higgs^2/(Y2Higgs−Y1Higgs)
47 // Y2 = Y2Higgs - Y3Higgs^2/(Y2Higgs−Y1Higgs)
48 // Y3 = 0
49 // Z1 = Z1Higgs − 4*(Y3Higgs/(Y2Higgs−Y1Higgs))*Z6Higgs
50 // Z2 = Z2Higgs + 4*(Y3Higgs/(Y2Higgs−Y1Higgs))*Z7Higgs
51 // Z3 = Z3Higgs + 2*(Y3Higgs/(Y2Higgs−Y1Higgs))*(Z6Higgs-Z7Higgs)
52 // Z4 = Z4Higgs + 2*(Y3Higgs/(Y2Higgs−Y1Higgs))*(Z6Higgs-Z7Higgs)
53 // Z5 = Z5Higgs + 2*(Y3Higgs/(Y2Higgs−Y1Higgs))*(Z6Higgs-Z7Higgs)
54 // Z6 = Z6Higgs + 4*(Y3Higgs/(Y2Higgs−Y1Higgs))*(Z1Higgs-Z3Higgs-Z4Higgs-Z5Higgs)
55 // Z7 = Z7Higgs + 4*(Y3Higgs/(Y2Higgs−Y1Higgs))*(Z3Higgs+Z4Higgs+Z5Higgs-Z1Higgs)
56 // Also, we have not included Y1 and Z1 as parameters since they are related to the SM parameters:
57 // Z1 = 2\lambda
58 // Y1 = -\mu^2
59 // being \lambda and \mu the parameters of the SM Higgs potential
