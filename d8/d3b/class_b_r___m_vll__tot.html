Detailed Description

A class for the binned observable \(<BR>\) in \(M \to V l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the binned observable \(<BR>\) in \(M \to V l^+l^-\) in terms of the binned CP averaged helicity coefficients \(<\Sigma_i>\), computed in the MVll class, and the meson width \(W_M\):

\[ <BR>= \frac {<\Gamma'>}{W_M} \,. \]
