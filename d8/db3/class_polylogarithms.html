Detailed Description

A class for the polylogarithms.

Author

This class handles the polylogarithms \(\mathrm{Li}_2(x)\), \(\mathrm{Li}_2(z)\) and \(\mathrm{Li}_3(x)\), where \(x\) and \(z\) are real and complex variables, respectively.
