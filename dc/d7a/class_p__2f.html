Detailed Description

A class for the binned observable \(P_2\) in \(M \to V l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the observable \(P_2\) in \(M \to V l^+l^-\) in terms of the CP averaged helicity coefficients \(\Sigma_i\), computed in the MVll class:

\[ P_2=\frac {\Sigma_6}{8\Sigma_{2s}}\,. \]
