Detailed Description

A module for the extension of either the Standard Model or the \(Z_2\) symmetric Two-Higgs-Doublet model by a scalar octet.

The Standard Model extension by a scalar octet can be accessed setting the THDMWmodel flag to "ManoharWise". A custodial limiting case of this model can be obtained setting the flag to "custodialMW". The THDM plus octet can be address by attributing "custodial1" to the THDMWmodel flag. The implemented observables are positivity, unitarity, Higgs signal strengths and electroweak precision measurements. (Not all observables are defined for all model types!)
