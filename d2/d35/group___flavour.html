Detailed Description

A module for all the flavour observables implemented in HEPfit.

A module for flavour observables.

This module has several flavour physics observables which include quark flavour violation in the beauty, charm and strange sectors. This includes the evolutors, Hamiltonians and low energy observables.

AmpDD2::AmpDD2 ( const StandardModel &  SM_i)

\( | \Delta C = 2 | \) Amplitude Class

This class is related to the calculation of the \( D^{0}-\bar{D}^{0}\) mixing amplitude in relation to the absorptive part of the mixing;
it is written to consider also the contribution coming from supersymmetric theories in the squark-gluino-quark sector in the mass insertion approximation.
