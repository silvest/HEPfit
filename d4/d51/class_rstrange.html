Detailed Description

An observable class for \(R_s^0=\Gamma(Z\to s\bar{s})/\Gamma(Z\to {\rm hadrons})\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the ratio of the \(Z\to s\bar{s}\) width to the \(Z\)-boson hadronic width:

\[ R_s = \frac{\Gamma_s}{\Gamma_h}\,. \]

See also
EW_NPZff::Rstrange() and the detailed description of EW class for the inclusion of new physics contribution
