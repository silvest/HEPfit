Detailed Description

A class for the observable \(\Gamma'\) in \(M \to V l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the observable \(\Gamma'\) in \(M \to V l^+l^-\) in terms of the CP averaged helicity coefficients \(\Sigma_i\), computed in the MVll class:

\[ \Gamma'=- \frac {1}{4} ((3\Sigma_{1c} - \Sigma_{2c}) + 2(3\Sigma_{1s} - \Sigma_{2s}))\,. \]
