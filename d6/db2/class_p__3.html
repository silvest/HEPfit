Detailed Description

A class for the binned observable \(<P_3>\) in \(M \to V l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the binned observable \(<P_3>\) in \(M \to V l^+l^-\) in terms of the binned CP averaged helicity coefficients \(<\Sigma_i>\), computed in the MVll class:

\[ <P_3>=- \frac {<\Sigma_9>}{4<\Sigma_{2s}>}\,. \]
