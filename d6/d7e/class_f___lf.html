Detailed Description

A class for the observable \(F_L\) in \(M \to V l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the observable \(F_L\) in \(M \to V l^+l^-\) in terms of the CP averaged helicity coefficients \(\Sigma_i\), computed in the MVll class, and the meson width \(M_W\):

\[ BR= \frac {3\Sigma_{1c}-\Sigma_{2c}}{4\Gamma'} \,. \]
